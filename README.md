# SchedulingSystem
Online scheduling system for beauty saloons

## Running the projects

### Frontend:

1. **Open the Front-End folder:**

```shell
cd frontend
```

2. **Start the application:**

```shell
npm start
```

3. **Access on the browser:**

- <http://localhost:3000>

### Backend:

1. **Open the Back-End folder:**

```shell
cd Backend
```

2. **Run the application:**

```shell
dotnet run --launch-profile "https"
```

3. **Test on the browser:**

- <https://localhost:7192/weatherforecast>
